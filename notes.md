### üß† Key concepts I learned (ROS 2 basics)
**Workspace (`ros2_ws`)**
- A folder that holds ROS packages and build outputs

**Package (`my_first_pkg`)**
- A ROS project that contains nodes + config files that ROS can recognize and run

**Node**
- A running program in ROS (publisher node, subscriber node, etc.)

**Topic**
- A named channel where messages flow (example: `/chatter`)

**Publisher**
- Sends messages to a topic

**Subscriber**
- Receives messages from a topic
